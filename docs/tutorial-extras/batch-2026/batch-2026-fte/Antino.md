# Antino Interview Experience

**Company Name**: Antino
**Job Description**: Software Development Trainee (SDE)

## Online Assessment Round

### 1) Online Test

**Mode:** Online

* The round consisted of **MCQ-based questions** along with **DSA problems**.
* **No. of MCQ Questions:** Yes (Basic CS/technical MCQs)
* **No. of DSA Questions:** 3 coding questions
* MCQs mainly covered **basic CS fundamentals and technical concepts**.
* This round focused on testing **problem-solving ability and core concepts**.

## Technical Interview Round

**Mode:** Virtual / Online

* The interview started with a **self-introduction**.
* I was asked to explain **one of my projects in detail**.

### Topics Covered

#### SQL Questions

* Basic to intermediate level SQL queries.
* Questions related to:

  * `SELECT`, `WHERE`, `JOIN`
  * `GROUP BY`, `HAVING`
  * Basic database concepts

#### CS Fundamentals

* OOPS basics
* Operating Systems fundamentals
* DBMS concepts
* Computer Networks basics

#### DSA Question

* 1 coding problem.
* I was asked to:

  * Explain the **approach**
  * Discuss **time and space complexity**

## HR Round

### Questions Asked

* A few **basic technical questions**.

## Final Note

* Prepare well for **MCQs + DSA** for the online assessment.
