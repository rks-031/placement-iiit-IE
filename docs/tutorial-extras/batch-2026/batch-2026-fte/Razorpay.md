# Razorpay Interview Experience  

**Article by:** Debashish Sahoo.  
**Role:** Software Development Engineer (Backend)  

---

## Round 1 — Online Assessment  

**Duration:** 1.5 hours  
**Content:**  
- **Topics:** Graphs, Disjoint Set Union (DSU), Trees  
- **Questions:** 2 medium-level and 1 hard-level coding question  

---

## Round 2 — Technical Interview (45 Minutes)

### Topics Discussed  

- **Introduction**  
  Brief self-introduction covering academics and projects.  

- **Internship Project (Dell Technologies – MCP Servers)**  
  Discussed:  
  - MCP server architecture  
  - JSON payload generation by LLM clients  
  - Schema validation and constraints  
  - Handling malformed/nested JSON  
  - Tool invocation logic  
  - End-to-end request–response workflow  
  - Processing tool calls inside the MCP server  

- **DSA Question**  
  Modified *Frog Jump* problem:  
  - Allowed jumps of 1 or any prime ending with 3  
  - Explained dynamic programming and optimized transitions  

- **OOPs Concept**  
  Explained compile-time and runtime polymorphism with examples.  

- **Segment Tree vs Fenwick Tree**  
  Compared:  
  - Use cases  
  - Time and space complexity  
  - When to choose each  

---

## Round 3 — Hiring Manager Round (30 Minutes)

### Discussion Areas  

- **Introduction**  
  Focused on learnings and growth during internship.  

- **Deep Dive into MCP Project**  
  Topics:  
  - MCP vs REST/WebSocket  
  - Tool-calling mechanism  
  - System architecture  
  - JSON schema enforcement and fixes  
  - External API/tool integrations  
  - Adding authentication and rate limiting  
  - Major challenges and solutions  
  - Ideas for improvement  

- **Behavioural Questions**  
  Collaboration, adaptability, and problem-solving.  

- **Leadership Scenario**  
  A situational question testing decision-making.  

- **Motivation & Fit**  
  Why Razorpay and alignment with engineering culture.  

- **Career & Learnings**  
  Experience with protocol design, schema validation, and LLM integration.  

---

## Pro Tips  

- Keep discussions interactive; share your thought process clearly.  
- Ask for clarification confidently when needed.  
- Research the company and interviewers before HR rounds.  
- Stay updated on product developments.  
  - Example: Razorpay’s custom MCP servers (RAY) for AI-driven payments.  
- For on-campus placements:  
  Remove anything from your resume that you cannot confidently justify.
