## Round 2 — Technical Interview 1

### DSA Discussion

* **Coding Question:**
  Sort an array of 1s and 0s in-place without using a third variable.
  Discussed brute force and optimized approaches.

### Core DSA Topics Covered

* Linked Lists vs Arrays
* Binary Search: implementation and use cases
* Stacks vs Queues
* Priority Queues, Min-Heap vs Max-Heap
* Binary Trees, B-Trees
* Hashing collisions, RSA hashing
* AVL trees and rotations
* Graph algorithms (BFS, DFS, Dijkstra, Bellman-Ford, Kruskal)
* Time and space complexities

### DBMS Questions

* Normalization levels and need
* ACID properties
* Inner join vs outer join
* Window and aggregate functions
* How BLOB data is stored in PostgreSQL
* SQL vs NoSQL
* Query: 3rd highest salary in each department

### CS Fundamentals

* OOP concepts: abstract classes, interfaces
* Method overloading vs overriding
* Constructors and destructors
* OS topics: multithreading, semaphores, mutex locks, IPC methods
* Virtual memory, page tables

### Project-related Questions

* Model architectures and workflows
* EfficientNetB0 choice and reasoning
* UNet architecture: follow-ups
* LSTM encoder–decoder project

  * RNN, GRU, embeddings, tokenization, NLP concepts
* Value addition and novelty in projects

### Domain-related Questions

None

### Internship Questions

* Responsibilities in past internships
* Contribution to company
* Leadership or conflict management experiences

## Round 3 — Technical Interview 2

### DSA Question

A full end-to-end binary tree exercise:

* Build a binary tree from scratch
* Insert and delete elements
* In-order traversal
* Convert to pre-order (recursion + iteration)
* Find maximum-sum path in the tree
* Modify insertion to convert tree into a BST
* Search for an element in the BST (return -1 if not found)
* Time and space complexity follow-ups

### DBMS Topics

* All levels of normalization
* ACID properties
* Partial, transitive, and functional dependencies
* Methods to resolve dependencies

### CS Fundamentals

None

### Project Discussion

None

### Domain-specific Questions

Covered everything:

* Full stack (React + Ruby on Rails)
* ML / DL engineering
* Automation
* BI tools

### Internship Discussion

Focused on SDE internship at BigBinary, reasons for choosing full stack, and motivation for an R&D role.
Leadership and team coordination scenarios were asked.

## Extra Notes

Only talk about things you genuinely understand. Anything on your resume is open for deep questioning.
Explain your reasoning clearly; interviews feel smoother when treated like a conversation instead of an exam.
HR rounds are usually relaxed and easygoing. Just keep it natural.
