## Round 2: Technical Interview

**Mode:** Virtual / Online

### Discussion Flow

* **Introduction**
  Brief self-introduction followed by questions on core computer science fundamentals.

* **DBMS**

  * Types of databases
  * Structured vs Unstructured data
  * Normalization and its types
  * One easy SQL query

* **Object-Oriented Programming (OOPS)**

  * Four pillars of OOPS with real-life examples
  * Types of polymorphism
  * Inheritance

* **Operating Systems**

  * Deadlock
  * Deadlock prevention

* **Data Structures & Algorithms (DSA)**

  * Search operation in a Binary Search Tree (BST)
  * Node deletion in a Linked List

* **Projects**

  * In-depth discussion of major project
  * Cross-questioning on design decisions
  * Implementation logic and technology choices
